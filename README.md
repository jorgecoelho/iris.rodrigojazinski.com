# iris.rodrigojazinski.com

Website for the **IRIS method** — Interpretive Rotation on Invariant Signals — a structured analytical framework developed by R. Jazinski for making visible how different interpretive frameworks process the same evidence into divergent conclusions.

## Site

**URL:** [iris.rodrigojazinski.com](https://iris.rodrigojazinski.com)

Single-page static site hosted on GitHub Pages. Four sections:

- **Method** — Formal description of IRIS, including the grammar definition G = (V, S, W, C, A)
- **Toolkit** — The IRIS Toolkit v5.1 (20-page operational guide, PDF download)
- **Case Studies** — Selected Analysis from the five-round validation programme (PDF download)
- **Archive** — Complete Archive hub document covering 150+ formatted entries across ~1,045,000 words of agent output (PDF download)
